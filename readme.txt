🌟 **CTC Rating** allows users to easily add a thumbs up 👍 and thumbs down 👎 rating system to your WordPress posts. Users can also view the total number of likes and dislikes, and even see who has liked or disliked a post in a sleek modal window.

== Description ==

CTC Rating is the perfect plugin to enhance user interaction on your WordPress site! 

🔹 **Thumbs Up/Thumbs Down:** Users can quickly express their opinion by giving a thumbs up 👍 or thumbs down 👎 to your posts.

🔹 **Display Counts:** Show the total number of thumbs up and thumbs down for each post, encouraging more interaction.

🔹 **View Users:** Let users see who has liked or disliked a post with a simple click, displayed in an elegant modal window.

🔹 **Gutenberg Block:** Seamlessly integrate the rating system into your posts with the Gutenberg block, making it easy to add the feature wherever you need it.

== Installation ==

1. Download the plugin from the WordPress plugin repository.  
2. Go to your WordPress dashboard, navigate to Plugins → Add New, and upload the downloaded file.  
3. Click "Activate" to start using CTC Rating.  
4. Add the rating block to your posts via the Gutenberg editor.
